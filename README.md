# 🚀 ChatDEVA — AI-Powered College Assistant (RAG System)

ChatDEVA is a **production-ready AI assistant for colleges**, built using **Retrieval-Augmented Generation (RAG)**.  
It enables institutions to upload academic documents and allows students to ask questions in natural language with **accurate, source-backed answers**.

## ✨ Features

### 🤖 AI Capabilities
- Natural language Q&A
- Context-aware answers using RAG
- Source-backed responses (document name, metadata)

### 📄 Document Management
- Upload PDF, DOCX, TXT files
- Automatic chunking & embedding
- Metadata tagging (filename, uploader, timestamp)

### 👥 User System
- Role-based access control:
  - **Admin** → full control
  - **Staff** → manage documents
  - **Student** → query system
- JWT-based secure authentication

### 🏫 Multi-College Support
- Multi-tenant architecture
- Complete data isolation per college
- Separate knowledge base per institution

### 💬 Chat System
- New chat sessions
- Persistent chat history per user
- Delete personal chats anytime

### 🛡️ Security & Control
- Secure APIs with FastAPI
- Audit logs for actions
- No unauthorized access to documents
